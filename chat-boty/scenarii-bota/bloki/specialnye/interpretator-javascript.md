# Интерпретатор JavaScript

### **Пример интерпретации кода JavaScript**



{% hint style="info" %}
Для установки используется специальная функция **setContactVariable**
{% endhint %}

```javascript
var name = "Watbot";
setContactVariable("name", "name")
```
